# Ad-Free TikTok & Instagram Downloader 🚀

Bu proje, TikTok ve Instagram içeriklerini (videolar, reels, fotoğraflar ve kaydırmalı fotoğraf gönderileri) **reklamsız** ve **doğrudan** indirebilmeniz için geliştirilmiş modern ve evrensel bir sosyal medya indirici uygulamasıdır. 

Next.js (App Router) kullanılarak geliştirilmiş olup, "Glassmorphism" (buzlu cam efekti) tasarımıyla şık, akıcı ve premium bir kullanıcı deneyimi sunar. Vercel üzerinden tek tıkla yayınlanmaya (deploy) hazırdır.

## Özellikler ✨

- **Tamamen Reklamsız:** Sinir bozucu açılır pencereler (pop-up) veya gizli reklamlar yok.
- **Akıllı Navigasyon (Menü):** Sağ üstteki modern hamburger menü ile TikTok ve Instagram sayfaları arasında hızlı geçiş imkanı.
- **TikTok İndirici:**
  - **Filigransız İndirme:** TikTok videolarını logolar ve yazılar olmadan saf MP4 formatında indirir.
  - **Fotoğraf Slayt Desteği:** Kaydırmalı fotoğraf gönderilerindeki tüm fotoğrafları tespit eder, ızgara (grid) halinde sunar ve tek tek yüksek kalitede indirmenize olanak tanır.
  - **Sesi İndirme:** Sadece videonun müziğini (MP3) indirebilme seçeneği.
- **Instagram İndirici:**
  - **Reels İndirme:** Yüksek kaliteli Reels videolarını doğrudan cihazınıza indirir.
  - **Fotoğraf ve Carousel Desteği:** Instagram gönderilerindeki tekil veya çoklu fotoğrafları otomatik olarak ayrıştırarak indirme butonlarıyla listeler.
  - **API Anahtarı Rotasyonu:** 5 farklı API key rotasyonu ile aylık yüksek limitlerde ücretsiz indirme kapasitesi.
- **Doğrudan Tarayıcı İndirmesi (Force Download):** Medyalar yeni sekmede açılmaz, arka plandaki Proxy API sayesinde doğrudan bilgisayarınıza veya telefonunuza inmeye başlar.
- **Karanlık Mod (Dark Mode):** Göz yormayan simsiyah şık tasarım.
- **Mobil Uyumlu (Responsive):** Telefonlardan ve tabletlerden kusursuz görünüm.

## Nasıl Kullanılır? 📱

1. TikTok veya Instagram uygulamasından/web sitesinden indirmek istediğiniz içeriğin **bağlantısını (linkini) kopyalayın**.
2. Sağ üstteki menü ikonuna tıklayıp indirmek istediğiniz platformu (TikTok veya Instagram) seçin.
3. Kopyaladığınız linki arama kutusuna **yapıştırın** ve **"İndir"** butonuna tıklayın.
4. Ekranda beliren indirme seçeneklerini (MP4, MP3 veya fotoğraflar) kullanarak medyayı anında cihazınıza kaydedin!

## Yerel Kurulum (Local Development) 🛠️

Projeyi kendi bilgisayarınızda çalıştırmak için:

1. Depoyu klonlayın:
   ```bash
   git clone https://github.com/kcbyk/ottiktok.git
   ```
2. Klasöre girin:
   ```bash
   cd ottiktok
   ```
3. Gerekli paketleri yükleyin:
   ```bash
   npm install
   ```
4. Geliştirme sunucusunu başlatın:
   ```bash
   npm run dev
   ```
5. Tarayıcınızda `http://localhost:3000` adresini açarak uygulamayı görüntüleyin.
